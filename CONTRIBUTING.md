# Contributing

This repository is **the public face** of Cross-Posting Superapp — README, screenshots, brand assets, integration docs. The product source itself is closed.

## What's accepted here

- Typo / clarity fixes in `README.md` / `README.ru.md`
- Documentation improvements in `/docs`
- Better screenshots / brand assets in `/img` (PR with PNG/SVG)
- Translations of `README.md` into new languages

## What's not

- Source code for the bot or Mini App — that lives in the private product repo
- Feature requests in this repo — open them at https://github.com/SMOService/cross-posting-superapp/issues with the `feature-request` label

## PR checklist

- [ ] Bilingual: if you change `README.md`, change `README.ru.md` accordingly (and vice versa)
- [ ] Image assets are optimised (TinyPNG / SVGO)
- [ ] No personal data in screenshots
