## Что делает

```
форвард в Telegram  →  выбор проекта (1 тап)  →  fan-out
                                                  ├──► Buffer       → X / LinkedIn / Threads / FB / IG / ...
                                                  ├──► Upload-Post  → TikTok / Reels / Shorts / VK Clips
                                                  └──► Postmypost   → VK / OK / Дзен / Threads / IG
```

Используются твои собственные API-ключи Buffer / Upload-Post / Postmypost. Бот оркестрирует fan-out, провайдеры держат OAuth с самими соцсетями.

---

## Зачем нужен

Если ты ведёшь больше одного бренда — бизнес, личный аккаунт, контент-мейкера на сопровождении — ты знаешь эту боль:

- Buffer хорошо работает с текстом и фото, но не с короткими вертикальными видео.
- Upload-Post покрывает TikTok / Reels / Shorts, но это отдельный кабинет.
- Postmypost закрывает RU-соцсети (VK / OK / Дзен), которые западные агрегаторы игнорируют.

**Ни один агрегатор не покрывает всё.** Строить ещё один агрегатор с нуля — месяцы работы. Cross-Posting Superapp — это **роутер** перед тремя агрегаторами, которые в сумме покрывают всё, организованный по **проектам**, чтобы один бот обслуживал твой бизнес, твой личный аккаунт и аккаунт клиента без утечек ключей между ними.

## Возможности

### Несколько проектов в одном боте
«Мой бизнес», «Личный», «Клиент X» — у каждого свои ключи провайдеров, свой набор каналов, свои правила маршрутизации. Переключение между проектами в один тап.

### Маршрутизация TEXT vs VIDEO
Какие-то соцсети живут текстом. Какие-то — короткими вертикальными видео. Каждый проект решает per-mode, куда идут посты — чтобы 30-секундное TikTok-style видео не приехало неуклюжей видео-ссылкой на LinkedIn.

### Workflow — это пересылка
Никаких форм загрузки. Никаких интерфейсов планирования. Пересылаешь сообщение в бот — он понимает тип медиа, спрашивает в какой проект, делает fan-out. Два тапа для публикации в 10+ соцсетей.

### Оплата Telegram Stars
Платишь Stars прямо в Telegram. Без банковских карт. Без cross-border трения с платежами. Отменить можно в любой момент — проекты остаются.

### Ключи encrypted at rest
Твои API-ключи Buffer / Upload-Post / Postmypost шифруются AES-256-GCM с ключом, уникальным для инстанса. Бот держит plaintext только в памяти на время одного fan-out вызова — никогда не на диске, никогда в логах.

## С чего начать

1. Открой **[@SuperappAIbot](https://t.me/SuperappAIbot)** в Telegram → `/start`
2. Тапни **Открыть Mini App** → создай первый проект
3. Вставь свои API-ключи Buffer / Upload-Post / Postmypost (шифруются при получении)
4. Выбери, в какие каналы этот проект публикует per-mode (TEXT / VIDEO)
5. Перешли любое сообщение в бот — готово

## Экосистема

Superapp — это слой **рассылки**. Остальное для ведения канала закрывают соседние боты — создать контент, достать медиа, перенести между каналами:

- **[@ZavodClawbot](https://t.me/ZavodClawbot)** — ИИ Контент-Завод: источники → ИИ пишет оригинальные посты в стиле канала и публикует по расписанию
- **[@TonChatAIbot](https://t.me/TonChatAIbot)** — AI-ассистент для текстов, картинок и идей ([tonchat.ai](https://tonchat.ai))
- **[@SaveVideoProbot](https://t.me/SaveVideoProbot)** — скачивание видео из TikTok / YouTube / Reels / VK для постов
- **[@BridgePostBot](https://t.me/BridgePostBot)** — копирование и AI-перевод постов между Telegram-каналами

Open-source кирпичики:

- **[Buffer Poster Bot](https://github.com/SMOService/buffer-poster-bot)** — self-hostable single-user предшественник (open source, MIT)
- **[Cross-Post Bridge AI bot](https://github.com/SMOService/Cross-Post-Bridge-AI-bot)** — AI-мосты между TG-каналами с рерайтом и переводом
